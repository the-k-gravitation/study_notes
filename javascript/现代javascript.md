## 数组 Array

`Array` 是有序集合。  数组元素是从 `0` 开始编号。

创建一个空数组：

```js {.line-numbers}
let arr = new Array();

let arr = []; // 这种方式用得比较多
```

获取数组中的元素：

```js {.line-numbers}
const months = [
  "January",
  "February",
  "March",
  "April",
  "May",
  "June",
  "July",
  "August",
  "September",
  "October",
  "November",
  "December"
];



```
